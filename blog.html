Question-1: What are the different ways to select an element in the DOM?

DOM থেকে কোনো এলিমেন্ট সিলেক্ট করার বিভিন্ন উপায় রয়েছে। এখানে কিছু জনপ্রিয় মেথড উল্লেখ করা হলো: getElementById() দিয়ে নির্বাচন, getElementsByClassName() দিয়ে নির্বাচন, getElementsByTagName() দিয়ে নির্বাচন, querySelector() দিয়ে নির্বাচন।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

এদের মধ্যে কিছু গুরুত্বপূর্ণ পার্থক্য রয়েছে। innerHTML HTML ট্যাগ সমর্থন করে। innerText HTML ট্যাগ ইগনোর করে, শুধুমাত্র দৃশ্যমান টেক্সট দেখায়। এবং textContent এলিমেন্টের সব টেক্সট রিটার্ন করে, তবে HTML ট্যাগ হিসেবে ব্যাখ্যা করে না।

Question-3: What is event delegation in the DOM?

Event Delegation হলো একটি Parent এলিমেন্টে ইভেন্ট যুক্ত করা, যাতে তার Child এলিমেন্টগুলোর ইভেন্টও হ্যান্ডেল করা যায়। এটি ডাইনামিক এলিমেন্ট হ্যান্ডেল করা, কোড অপটিমাইজ করা, এবং পারফরম্যান্স উন্নত করতে ব্যবহৃত হয়।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো একটি ইভেন্ট যখন চাইল্ড এলিমেন্টে ট্রিগার হয়, তখন সেটি তার প্যারেন্ট, গ্র্যান্ডপ্যারেন্টসহ উপরের দিকে root element পর্যন্ত ছড়িয়ে পড়ে। এটি ইভেন্ট হ্যান্ডলিং সহজ করতে ব্যবহৃত হয়।

Question-5: How do you create, add, and remove elements using JavaScript?

নতুন এলিমেন্ট তৈরি করতে document.createElement() ব্যবহার করা হয়, যা DOM-এ appendChild(), append() দিয়ে যোগ করা যায়। এলিমেন্ট মুছে ফেলার জন্য remove() বা removeChild() ব্যবহার করা হয়। এসব মেথড ব্যবহার করে সহজেই DOM-এ এলিমেন্ট যুক্ত, সংশোধন ও মুছে ফেলা সম্ভব।
